# 🎵 Spotify Insights Dashboard

A complete end-to-end Power BI data analysis project
exploring global Spotify music trends across 17.55K+
tracks using Power BI and Microsoft Excel.

---

## 🛠️ Technologies

- Power BI Desktop
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Dataset: Spotify Music Data

---

## 📊 Features

Here's what this dashboard analyzes:

- **Total Streams**: What is the overall streaming
  volume across all tracks?

- **Platform Distribution**: How is content split
  between Spotify (80%) and YouTube (20%)?

- **Top Artists**: Which artists lead in total
  streams and engagement?

- **Album Type Analysis**: How do Singles, Albums
  and Compilations compare in streams?

- **Licensed Content**: What percentage of streamed
  content is officially licensed?

- **Total Likes & Views**: What is the engagement
  level across tracks?

## ⚙️ The Process

I started by collecting the Spotify dataset and
importing it into Power BI via Excel.

The data required cleaning, null handling, and
proper data type formatting before any visuals
could be built.

I then built 6+ visuals progressively —
from KPI cards to bar charts and donut charts —
to tell a complete story about music streaming trends.

---

## 🧠 What I Learned

### 🔴 Data Modeling
- **Relationships**: Connected tables correctly
  for cross-filtering between slicers and visuals
- **DAX Measures**: Built custom measures for
  Total Streams, Likes, Views, and Engagements

### 🔗 Slicer Interactions
- **Multi-level filtering**: Album, Artist, and
  Track slicers filter all visuals dynamically
- **Cross-filtering**: Visuals respond to each
  other's selections

### 📊 Visualization Skills
- **Chart selection**: Chose right chart types
  for each insight (bar, donut, KPI cards)
- **Design thinking**: Dark theme with green
  accents for Spotify brand consistency

### 🛠️ Dashboard Setup
- **Full workflow**: From raw CSV to published
  interactive dashboard
- **Layout design**: Balanced KPIs, charts and
  text insights on one screen

## 🚀 Running the Project

1. Download the Spotify dataset (CSV)
2. Open Power BI Desktop
3. Import CSV via Get Data → Text/CSV
4. Apply transformations in Power Query
5. Build relationships in Model view
6. Run DAX measures and build visuals
7. Apply dark theme and publish
